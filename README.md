# mike-t-302f-test-website-002

Next.js App Router project initialized with TypeScript.

## Development

Install dependencies:

```bash
npm install
```

Run the development server on `0.0.0.0:8080`:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

The production build is a static export emitted to `out/`.

Serve the static bundle on `0.0.0.0:8080`:

```bash
npm run serve
```
